PathToBonjwa

Brood War Replay Analyzer

Download for Windows

v0.1.0 — Phase 1 MVP (10.3 MB)

Features

  • Parse BW .rep replay files — players, races, map, matchup
  • Game duration, timestamp, speed, game type
  • Player colors matching in-game colors
  • Drag-and-drop or file browser to load replays
  • Dark theme, single portable .exe, zero dependencies
